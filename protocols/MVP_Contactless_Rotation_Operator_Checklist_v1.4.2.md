# MVP Contactless Rotation Operator Checklist v1.4.2

**Protocol ID:** SEG-MVP-ROT-001  
**Use with:** `MVP_Contactless_Rotation_Protocol_v1.4.2.md`

## Before Testing

- [ ] Read `START_HERE.md`.
- [ ] Read `safety/High_Voltage_Risk_Register.md`.
- [ ] Confirm no high-voltage system is active during this MVP motion test.
- [ ] Confirm rotor guard is installed.
- [ ] Confirm exclusion zone is marked.
- [ ] Confirm emergency stop is available.
- [ ] Complete build sheet.
- [ ] Complete conditioning sheet.
- [ ] Confirm magnet groups A, B, and C.
- [ ] Assign blind codes where practical.
- [ ] Save randomized trial order.
- [ ] Complete before-test field maps.
- [ ] Confirm fixed gap jig.
- [ ] Confirm rotor runout measurement.
- [ ] Confirm motor warm-up time.
- [ ] Confirm camera, power, temperature, and RPM logging.

## During Testing

- [ ] Start video and data logging before each trial.
- [ ] Use the randomized order.
- [ ] Use the same gap and start orientation.
- [ ] Do not hand-hold the magnet during formal trials.
- [ ] Record anomalies immediately.
- [ ] Stop if vibration, contact, overheating, smoke, arcing, or unsafe motion occurs.
- [ ] Do not interpret results during the trial block.

## After Testing

- [ ] Complete trial log.
- [ ] Save raw data files.
- [ ] Save videos and field maps.
- [ ] Generate SHA-256 hashes.
- [ ] Complete evidence manifest.
- [ ] Complete reviewer signoff.
- [ ] Update claim matrix only if evidence supports it.

## Claim Boundary

A successful run may support only the contactless-rotation conditioning claim. It does not validate energy generation, cooling, weight loss, antigravity, flight, propulsion, or over-unity operation.
