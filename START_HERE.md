# START HERE

This package is a V&V framework for SEG, the Searl Effect Generator. It is built for a reviewer, builder, or agent starting from zero context.

## Do This First

1. Read `QUICKSTART.md`.
2. Read `AGENT_BRIEF.md`.
3. Do not claim the SEG works.
4. Open the MVP protocol and operator checklist.
5. Create a real run folder under `results/runs/`.

## First Action

Do not run energy, cooling, weight, propulsion, or high-voltage tests first. Run only the MVP contactless-rotation protocol first.

## Do Not Claim Yet

Do not claim over-unity energy, ambient-energy extraction, self-running operation, cooling, weight loss, antigravity, flight, anomalous propulsion, or independent replication.

## Where Results Go

Create a run folder:

```text
results/runs/RUN-YYYYMMDD-001/
```

You may copy the layout from:

```text
results/runs/RUN-TEMPLATE/
```

Each real run folder must include raw data, completed logs, file hashes, evidence manifest, and reviewer signoff before claim status changes.
