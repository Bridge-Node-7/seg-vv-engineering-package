# Contributing

Contributions are welcome when they improve evidence discipline, safety, clarity, or reproducibility.

## Acceptable Contributions

- Better protocols and controls.
- Completed evidence packages.
- Field-map methods.
- Data-analysis improvements.
- Documentation corrections.
- Safety clarifications.
- Replication reports, including null results.

## Not Acceptable

- Breakthrough claims without raw evidence.
- Over-unity, antigravity, weight-loss, cooling, propulsion, or flight claims without protocol-matched data.
- High-voltage construction guides or discharge schematics.
- Personal contact information or unnecessary biographical details.
- Hype language or personal attacks.
- Claims based only on uncontrolled video impressions.

## Evidence Submission Rule

Use a run folder under:

```text
results/runs/RUN-YYYYMMDD-XXX/
```

Include completed logs, instrument notes, evidence manifest, file hashes, and reviewer signoff.

## Claim Matrix Rule

Do not move claim status upward unless the evidence package supports the change and reviewer signoff is complete.
