# Phased Validation Roadmap

## Phase 0: Documentation Freeze

Goal: publish stable, source-labeled V&V documentation.

Exit criteria:

- README published.
- Agent brief published.
- Claim matrix published.
- Safety register published.
- MVP protocol and checklist published.
- Templates and evidence manifest published.

## Phase 1: MVP Contactless Rotation

Question: Does conditioned magnetic patterning create repeatable, measurable behavior near spinning copper compared with controls?

Required evidence:

- Control magnet data.
- Sham-conditioned magnet data where possible.
- Conditioned magnet data.
- Fixed gap.
- Rotor RPM data.
- Video or motion tracking.
- Field maps.
- Power and temperature logs.

Exit criteria:

- Repeatable difference demonstrated, or null result documented.
- Raw data attached.
- Reviewer signoff complete.
- Claim matrix updated only if evidence supports it.

## Phase 2: Field Pattern Repeatability

Question: Can the conditioning method reliably produce intended magnetic field patterns?

Exit criteria: field patterning is repeatable, partially repeatable, or not repeatable under documented conditions.

## Phase 3: Stability and Rotor Dynamics

Question: What geometry and operating conditions produce stable motion without unsafe dynamics?

Exit criteria: safe operating envelope defined or testing paused.

## Phase 4: Energy Balance Gate

Question: Does any partial system produce electrical output that exceeds measured inputs, stored energy changes, and uncertainty?

Exit criteria: no excess found, or excess candidate requires independent replication. Do not use energy-generation language without this phase.

## Phase 5: Thermal / Cooling Gate

Question: Does the system produce cooling beyond ordinary thermal and measurement artifacts?

Exit criteria: thermal behavior explained, null, or anomaly candidate requiring replication. Do not use cooling language without this phase.

## Phase 6: Mass / Weight Gate

Question: Does the system produce a measurable apparent weight change under controlled conditions?

Exit criteria: no effect, artifact explained, or anomaly candidate requiring independent replication. Do not use weight-loss or gravity language without this phase.

## Phase 7: Independent Replication

Question: Can another operator or lab reproduce a result using the published protocol?

Exit criteria: replicated, failed to replicate, or inconclusive.

## Stop / Pivot Rules

- If Phase 1 fails, revise the conditioning hypothesis. Do not claim conditioned-contactless superiority.
- If Phase 2 fails, stop claiming repeatable field patterning.
- If Phase 3 identifies unsafe dynamics, stop rotation testing until controls are revised.
- If Phase 4 fails, do not use energy-generation or excess-output language.
- If Phase 5 fails, do not use cooling or refrigeration language.
- If Phase 6 fails, do not use weight-loss, gravity, or propulsion language.
- If Phase 7 fails, do not claim independent replication.

A null result is not a failure of the V&V system. It is a successful evidence result.
