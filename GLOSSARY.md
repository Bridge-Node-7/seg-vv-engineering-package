# Glossary

## SEG

Searl Effect Generator. This is the device/program term used in this package.

## V&V

Verification and validation. In this package, V&V means turning claims into controlled tests with evidence requirements.

## MVP

Minimum viable protocol. The smallest useful first test that can produce bounded evidence.

## SEG-MVP-ROT-001

The first-gate protocol for contactless rotation testing near a spinning copper rotor.

## Claim Matrix

The document that separates claims and tracks their evidence status. Do not update it from narrative alone.

## Evidence Readiness Level, ERL

A maturity ladder for claims:

- **ERL-0:** Narrative claim only.
- **ERL-1:** Informal or builder-reported observation.
- **ERL-2:** Controlled internal test.
- **ERL-3:** Quantified internal repeatability.
- **ERL-4:** Independent replication.
- **ERL-5:** Multi-site validation with error budgets and external review.

## Field Map

A measured record of magnetic-field structure before or after conditioning. Visual film alone is not enough for a strong claim.

## Conditioning

A source-reported process intended to change or pattern magnet behavior. In this package, conditioning is a hypothesis until controlled evidence supports it.

## Sham-Conditioned Magnet

A control magnet handled like a conditioned magnet but without the claimed active conditioning step. This helps reduce expectation and handling bias.

## Evidence Manifest

A file list for a run package. It records file names, claim IDs, trial IDs, instruments, hashes, storage locations, and reviewer notes.

## SHA-256 Hash

A cryptographic fingerprint for a file. It helps show that evidence files have not changed after logging.

## Reviewer Signoff

A documented review that confirms the run folder is complete enough to support a claim-status decision.

## Null Result

A result showing no meaningful effect. A null result is useful evidence and should be preserved.

## Control Magnet

A magnet used as a normal comparison sample. It should not receive the special conditioning being tested.

## Rotor Runout

The amount a spinning rotor wobbles or deviates from true rotation. Excess runout can create false motion effects or changing gaps.

## Measurement Artifact

A false or misleading result caused by the test setup, environment, sensor limits, camera aliasing, vibration, airflow, friction, or other non-SEG causes.
